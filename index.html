About Kevin Schmalz

I use data, technology and practical experience to add value and solve problems.


My interests are in combining software engineering, geospatial information systems(GIS) and computer aided design(CAD) for the development of data-focused solutions for the architecture, engineering and construction industries.


I'm currently exploring web-map development with LeafletJS and CesiumJS while improving my HTML, CSS and Javascript.


For data analysis I use Python and the following libraries: Pandas, GeoPandas, GDAL, Numpy, Matplotlib and more. My SQL and database experience is in PostgreSQL running PostGIS and Google BigQuery.


FAA Certified UAV pilot under Part 107.


Commercial software toolkit: Bentley MicrostationCE, GE Smallworld, ESRI ArcGIS, Autodesk Civil3D, Microsoft Office, Adobe Creative Suite, Agisoft Metashape


Open source software toolkit: Visual Studio Code, QGIS, FreeCAD, Blender, Meshroom, Cloud Compare, OpenDroneMap
